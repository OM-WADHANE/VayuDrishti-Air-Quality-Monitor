# VayuDrishti – Air Quality Monitoring App 🌫️

VayuDrishti is a web application that displays real-time Air Quality Index (AQI) and pollutant levels for cities worldwide.

## Features

• Real-time AQI monitoring
• Pollutant concentration display (PM2.5, PM10, CO, NO2, O3)
• AQI health recommendations
• City search functionality
• Location-based AQI detection
• Dark purple gradient UI

## Technologies

HTML
CSS
JavaScript
OpenWeather Air Pollution API
